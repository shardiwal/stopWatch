StopWatch - The jQuery timer

It's a jQuery plug-in. :)

API Documentations:

Initilization

`$('.stopwatch').stopWatch();`

Initilization with options

`$('.stopwatch').stopWatch( { 'format': 'MM:SS' } );`
`$('.stopwatch').stopWatch( { 'format': 'HH:MM:SS' } );`

Start timer

`$('.stopwatch').stopWatch('start');`

Stop timer

`$('.stopwatch').stopWatch('stop');`

Reset timer

`$('.stopwatch').stopWatch('reset');`

Get taken time

`$('.stopwatch').stopWatch('timetaken');`

Callbacks onStart, onStop and onRest

`$('.stopwatch').stopWatch( {`
    `'format': 'MM:SS',`
    `'onStart': function(){`
        `alert('Called on onstart');`
     `},`
    `'onStop' : function(){`
        `alert('Called on onstop');`
     `},`
    `'onReset': function(){`
        `alert('Called on onreset');`
     `},`
`} );`
